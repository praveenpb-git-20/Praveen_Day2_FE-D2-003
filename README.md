# FlowTrack 360 – Responsive Website

## Project Overview

FlowTrack 360 is a modern and responsive workforce management website developed using HTML5 and CSS3. The platform is designed to help organizations streamline employee management, attendance tracking, payroll processing, and productivity monitoring through a simple and professional user interface.

The project demonstrates responsive web design principles and modern front-end development practices.
